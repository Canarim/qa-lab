# qa-lab

Laboratório de estudo voltado a aprender e praticar TypeScript e Playwright.

## Stack

TypeScript, Vitest e Playwright.

## Como rodar

```bash
npm install
npm test             # roda os testes uma vez
npm run test:watch   # roda em modo watch
```
